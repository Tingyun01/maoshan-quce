## 🔴 紧急问题：首页吸引力不足

**现状诊断**：
- 首页是纯列表式布局（推荐横滚 + 双列网格），第一眼感觉就是"选择题合集"
- 导览图和秘境奇谈埋在推荐区末尾，不显眼，容易被忽略
- 缺少"哇！这是什么？"的第一印象
- 打开后像工具，不像游戏

**核心问题**：**用户打开第一屏看不到「游戏里的游戏」，只看到一堆测试列表。**

## 🎮 V2.0 — 「碟中谍·画中画」（备案通过后第一批上线）

> **核心理念**：测试是壳，游戏是核。用户以为是测试，玩着玩着发现是小游戏。

### 2.1 捉鬼 · 茅山道士（场景冒险第3章）

**入口**：通过导览图 → 元符万宁宫 → "夜巡捉鬼" 关卡进入

**玩法**：
```
场景：深夜荒村，你作为茅山弟子持符巡逻
每道题 = 遇到一只鬼
每道选项 = 选择应对方式（符箓驱鬼 / 桃木剑斩 / 咒语封印 / 机智周旋）

选项背后有隐藏属性：
- 道行值（决定最终战斗力）
- 功德值（决定最终善果）
- 胆魄值（决定是否被吓退）

结局：根据三个属性的高低，判定你是：
- 捉鬼天师（道高功深）
- 降魔真人（胆识过人）
- 善缘道士（功德圆满）
- 仓皇弟子（被鬼吓跑）
```

**视觉**：深蓝色调，鬼火飘浮背景，选项卡片用符纸样式

### 2.2 画符 · 朱砂落笔（微交互场景冒险第4章）

**入口**：通过导览图 → 德祐观 → "符箓考核" 关卡进入

**玩法**：
```
你不是选ABCD，而是用手指在屏幕上"画"——
屏幕显示半透明的符箓底图，你需要顺着轨迹描画
系统用 touchmove 轨迹检测"是否按顺序走过关键节点"

每道符对应一道测试题的内核：
- 驱邪符 = 你的勇气测试
- 招财符 = 你的欲望测试
- 平安符 = 你的忧虑测试
- 桃花符 = 你的情感测试

画的"质量"（通过率）也影响结果
```

**视觉**：黄纸朱砂风，Canvas实时绘画轨迹

### 2.3 灵猴谷 · 猴子测试（场景冒险第5章）

**入口**：导览图 → 灵猴谷（当前已有预告）

**玩法**：
```
5种猴子类型互动场景：
- 给猴子投喂（测慷慨度）
- 和猴子对视（测勇气值）
- 模仿猴子动作（测开放度）
- 猴子偷你东西的反应（测冷静度）

每个选择累积"猴子好感"，最终匹配你是哪种猴子
猴子类型图谱：猴王/机灵鬼/懒猴/捣蛋王/害羞猴
```

## 🧩 V2.5 — 深度游戏化（备案通过后第二批）

### 3.1 炼丹炉 · 合成系统

**概念**：每完成一次测试，获得一种"药材"（随机）。集齐特定组合可"开炉炼丹"。

```
药材类型：人参/灵芝/朱砂/雄黄/鹿茸/雪莲/龙涎/麝香

合成配方：
- 人参+灵芝+朱砂 = 筑基丹（解锁隐藏测试）
- 雪莲+龙涎 = 驻颜丹（解锁新头像框）
- 雄黄+鹿茸 = 驱邪丹（解锁捉鬼场景）

每天最多得1味药材 → 鼓励每日回来 → 收集驱动留存
```

### 3.2 全服排行榜（社交驱动）

```
- 测试完成数量榜
- 收集成就最多榜
- 稀有结果持有榜（如"测出财神转世的只有3%"）
- 每日道签运势PK
```

### 3.3 个性化主页（道场系统）

```
用户完成5次测试后，解锁「我的道场」
- 显示你的守护神兽（根据测试结果）
- 展示你的五行属性条
- 你的"仙途等级"（修炼 → 筑基 → 金丹 → 元婴 → 化神）
- 等级由完成测试数+天数累积
```
